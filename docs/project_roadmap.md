# MemoryMap Project Roadmap

## Goal
Build a modern full-stack memory journaling and social mapping platform within 1 month by focusing on core scalable features first.

---

# Month Plan

## Week 1 — Foundation
### Goals
- Setup Next.js project
- Setup PostgreSQL database
- Configure Prisma ORM
- Configure project structure
- Learn Next.js App Router basics
- Setup Tailwind CSS
- Create base layouts

### Deliverables
- Running Next.js app
- Database connection working
- Initial schema created

---

## Week 2 — Authentication + Memories
### Goals
- JWT Authentication
- Signup/Login APIs
- HTTP-only cookie sessions
- Create Memory API
- Memory creation form
- Cloudinary image upload

### Deliverables
- User authentication working
- Memory creation working

---

## Week 3 — Maps + Profiles
### Goals
- Integrate Leaflet maps
- Render memory pins
- Create profile pages
- Public/private visibility filtering
- Timeline page

### Deliverables
- Interactive map working
- Profiles functional

---

## Week 4 — Social Features + Polish
### Goals
- Followers system

- Deployment
